ℹ️ Data Source Instructions

📊

Usage Data

You can obtain GitHub Copilot usage data from your enterprise insights page:

https://github.com/enterprises/{your-enterprise-name}/insights/copilot

Navigate to this page and export the usage data as JSON or JSONL format.

👥

User Data

You can export user data from your enterprise people management page:

https://github.com/enterprises/{your-enterprise-name}/people

Go to this page and export the user list as CSV format.
